HTML 标签

基本语法概述

  • HTML 标签是由尖括号包围的关键词，如 <html>。
  • HTML 标签通常是成对出现的，如<html>和</html>，我们称之为双标签。标签对中的第一个标签是开始标签，第二个是结束标签。
  • 单标签：情况极少，如<br />，我们称之为单标签。

标签关系

双标签关系可以分为两类：包含关系和并列关系

包含关系：（可以理解为父子关系）

并列关系：（可以理解为兄弟关系）

HTML 基本结构标签

第一个HTML网页

文档类型声明标签

<!DOCTYPE html>

lang语言种类

  • en定易语言为英语
  • zh-CN定义语言为中文
  • 简单来说en为英文网页，zh-CN为中文网页

字符集

字符集(Character set)是多个字符的集合，以便计算机可以识别和存储文字。

在<head>标签内可以通过<meta>标签的charset属性来规定HTML文档应该使用那种字符编码。

charset常用的值有：GB2312、BIG5、GBK和UTF-8，其中UTF-8被称为万国码，基本包含了全世界所有国家需要的字符。

HTML 常用标签

标题标签 <h1> - <h6>

这个标签可以使王爷具有语义化。HTML 提供了6个等级的网页标，即<h1> -<h6>。

单词head的缩写，意为标题、头部。在<body> - </body>里面使用。

标签语义：作为标签使用，并且依据重要性递减。

特点：

  • 加了标题的文字会变粗，字号也会依次变大
  • 一个标题独占一行

段落标签和换行标签

段落标签：<p> - <p>,标签用于定义段落，可以将网页分为若干段落。

单词paragraph的缩写，意为段落。

标签语义：可以把HTML 文档分割为若干段落。

特点：

  • 文本会根据浏览器的大小自动调整换行
  • 段落和段落之间保有空隙。

换行标签：<br />,可以是文本强制换行显示。

单词break的缩写，意为打断、换行。

标签语义：强制换行。

特点：

  • <br />是一个单标签。
  • <br />只是简单的换行，与段落不一样，段落之间会加入一些垂直的间距。

文本格式化标签

标签语义：突出重要性，比普通文字重要

<div>和<span> 标签

<div>和<span>是没有语义的，就像是一个盒子，用来装内容的。

div是division的缩写，表示分割、分区。span意为跨度、跨距。

特点：

  • <div>标签用来布局，但是一行只能放一个<div>。大盒子
  • <span>标签用来布局，一行上可以多个<span>。小盒子

图像标签和路径

图像标签

<img>标签用于定义HTML 页面中的图像。

单词image的缩写，意为图像。

src是<img>标签的必须属性（简单理解，就是图像标签的特性），他用于指定图像文件的路径的文件名。

注意：

  • 图像标签可以拥有多个属性，必须写在标签名后边。
  • 属性之间不分先后顺序，标签名与属性、属性与属性之间均已空格分开。
  • 属性采取键值对的格式，即key=”value”的格式，属性=“属性值”。
路径

相对路径：以引用文件所在的位置为参考，而建立出的目录路径，简单说：图片图片相对于HTML 页面的位置。

这里说的上一级、下一级、同级都是图片相对于HTML 页面的位置。

绝对路径：指目录下的绝对位置，直接到达目标位置，通常从盘符开始的路径或者是网站。

超链接标签

在HTML标签中，<a>标签用于定义超链接，作用是从一个页面链接到另一个页面

语法规范

单词anchor的缩写，意为“锚”。

两个属性的作用如下：

链接分类

外部链接：如<a href=”123456879”> 百度 </a>

内部链接：网站内部页面之间的链接，直接链接内部页面名称即可，如<a href=”index.html”>首页</a>。

空链接 ：<a href=”#”>首页</a>

下载链接：<a href=”img.zip”>下载文件</a>

网页元素链接：在网页中的各种网页元素，如文本、图像、表格、音频、视频都可添加超链接。

锚点链接：点击链接，可以快速定位到也买你中的某个位置

在链接文本的href属性中，设置属性值为#名字的形式，如<a href=”#2”>The second</a>

找到目标位置标签，里边添加一个id属性=刚才的名字，如：<h3 id=”2”>The secong</a>

HTML 中的注释和特殊字符

注释

在文档里添加一些便于阅读和理解但又不需要在页面中的显示的文字，需要注释标签。

HTML中的注释以”<!–”开头，以”–>”结束。

特殊字符

在HTML中，一些特殊的字符很难或者不便直接使用，此时需要我们用以下字符替代。

表格标签

表格主要的作用

表格主要用来显示、展示数据。表各规整、可读性非常好。

表格的基本语法

  • <table> - </table> 是用于定义表格的标签
  • <tr> - </ tr>标签用于定义表格中的行，必须嵌套在<table> - </table>中。
  • <td> - <td>用于定义表格中的单元格，必须嵌套在<tr> - </ tr>。
  • 字母td指表格数据（table data），即数据单元格的内容。

表头单元格标签

一切表头单元格标签位于表格的第一行或第一列，表头单元格里的文本内容加粗居中显示。

<th>标签表示表格的表头部分（table head的缩写）

表格属性

开发中不常用，后期用CSS搞他。

表格结构标签

为了使表格的语义更好，可以将表格分割为表格头部和表格主体两大部分

在表格标签中，分别用：<thead>标签表示头部区域、<tbody>标签表示表格的主体区域，可以清楚的分清表格结构。

  • <thead> </ thead>：用于定义表格的头部。<thead>内部必有<tr>标签。一般是位于第一行。
  • <tbody> </ tbody>：用于定义表格的主体，主要用于放数据本体。

合并单元格

  • 合并单元格的方式：

    • 跨行合并：rowspan=“合并单元格个数”
    • 跨列合并：colspan=”合并单元格个数”

  • 目标单元格：（写合并代码）

    • 跨行：最上测单元格为目标单元格，写合并代码
    • 跨列：最左侧单元格为目标单元格，写合并代码

  • 合并单元格三步曲

    • 首先，确定跨行还是跨列
    • 其次，找到目标单元格，写上合并的方式=合并的数量。如：< td colspan=”2”> </td>.
    • 删除多余的单元格。

列表标签

列表是用来布局的。

列表的最大特点就是整齐、整洁、有序，他作为布局会更加的自由和方便

分局使用情景不同，列表可以分为三大类：无序、有序和自定义。

无序列表

<ul>标签表示无序列表，一般会以项目符号呈现列表项，列表项使用<li>定义标签。

注意：

  • 无序列表的各个列表之间没有顺序级别之分，是并列的。
  • <ul></ul>里边只可以放<li></ li>,其他标签不允许。
  • <li></li>之间相当一个容器，可以容纳多种元素。
  • 无序列表会有自己的样式属性，但在实际应用中，会用CSS设置。

有序列表

<ol>标签表示有序列表，列表排序以数字来显示，列表项使用<li>定义标签。

  • <ul></ ul>里边只可以放<li></ li>,其他标签不允许。
  • <li></li>之间相当一个容器，可以容纳多种元素
  • 有序列表会有自己的样式属性，但在实际应用中，会用CSS设置。

自定义列表

自定义列表常用于对属于或者名词进行解释和描述，定义列表的列表项前没有任何项目符号。

在HTML标签中，<dl>标签用于定义描述列表，该标签会与<dt>（定义项目/名字）和<dd>（描述每一个项目/名字）一起使用。

  • <dl></dl>里边只能包含<dt></dt>和<dd></dd>。
  • <dt>和<dd>个数没有限制，经常是一个<dt>对多个<dd>。

表单标签

表单的作用

收集用户信息，像是个人简介就属于表单。

表单的组成

在HTML 中，一个完整的表单通常由表单域、表单控件（表单元素）和提示信息三部分组成。（不厚道的笑了）

表单域

表单域是一个包含表单元素的区域。

在HTML标签中，<form>标签用于定义表单域，以实现用户信息的收集和传递。

<form>会把他范围内的表单元素信息交给服务器。

表单元素

<input>表单元素

input输入的意思，而在表单元素中<input>标签用于收集用户信息。

  • input标签属于单标签
  • type属性设计不同的属性值用来指定不同的控件类型。

input的其他属性值：

  • name和value是每个表单元都有的属性值，主要给后台人员使用。
  • name表单元的名字，要求单选按钮和复选框要有相同的name值。
  • checked属性主要针对单选按钮和复选框，主要作用是打开页面，就可以默认选中某个表单元素。
  • maxlength属性的作用是在表单元素输入的最大字符数。
<lable>标签

<lable>标签为input元素定义标签。

<lable>标签用于绑定一个表单元素，当点击<lable>标签内的文本时，浏览器会自动将焦点转到或者选择对应的表单元素上，用来增加用户体验。

核心：<lable>标签的for属性应当与无关元素的id属性相同。

<select>标签

在页面中，如果有多个选项让用户选择，并且想要节约页面的空间时，我们可以使用<select>标签元素定义下拉列表。

  • <select>中至少包含一对<option></option>。
  • 在<option>中定义selected=“selected”时，当前项即为默认选中项。
<textarea> 表单元素

使用场景：用户输入的内容太多的时候，不使用文本框而改用<textarea> 标签。

在表单元素中，<textarea> 标签是用于定义多行文本输入的控件，文本域元素。
