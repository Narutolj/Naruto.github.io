Bootstrap

1、概念

Bootstrap是美国Twitter公司的设计师Mark Otto和Jacob Thornton合作基于HTML、CSS、JavaScript开发的简洁、直观、强悍的前端开发框架，使得 Web 开发更加快捷。Bootstrap提供了优雅的HTML和CSS规范，它即是由动态CSS语言Less写成。

  • 框架：一个半成品的软件，开发人员可以在框架的基础上进行开发，简化代码

  • 好处

    • 定义了很多的css样式和js插件，开发人员可直接使用

    • 响应式布局

      • 同一套页面可以兼容不同分辨率的设备

2、快速入门

  • [下载][https://www.bootcss.com/]
  • 在项目中将三个文件夹复制到
  • 创建html页面，引入必要的资源文件

3、BootStrap模板：

响应式布局

  • 同一套页面可以兼容不同分辨率的设备
  • 实现：依赖于栅格系统–将一行平均分为12个格子，可以指定元素占几个格子
  • 步骤：
    1. 定义容器。相当于之前的table
      • 容器分类
        1. container：两边留白
        2. container-fluid：每一种设备都是100%宽度
    2. 定义行。相当于之前的tr 样式：row
    3. 定义元素。指定该元素在不同的设备上，所占的格子数目。样式：col-设备代号-格子数目

  • 注意事项
    1. 一行中如果格子数目超过12，则超出部分自动换行
    2. 栅格类属性可以向上兼容。栅格类适用于与屏幕宽度大于或等有分解点大小的设备。
    3. 如果真实设备的宽度小于了设置杀个类属性的设备代码的最小值，会一个元素沾满一整行

CSS样式和JS插件

全局CSS样式

  1. 按钮：class=”btn btn-default”
  2. 图片：class=”img-responsive”：图片在任意尺寸都是100%
    • 图片形状
      1. <img src=”…” alt=”…” class=”img-rounded”> 方形
      2. <img src=”…” alt=”…” class=”img-cricle”> 圆形
      3. <img src=”…” alt=”…” class=”img-thumbnail”> 相框
  3. 表格：
    1. table
    2. table-bordered
    3. table-hover
  4. 表单
    1. 给表单添加：class=”form-control”
