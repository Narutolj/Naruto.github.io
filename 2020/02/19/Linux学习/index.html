Linux 学习

基础知识

文件与文件夹

文件：一般是一个独立的东西，可以通过一些特定工具及进行打开，并且其中不能包含除了文字以外的东西。

文件夹：可以包含其他东西的东西

目录结构

  • bin:全称binary,含义是二进制，该目录中存储的都是一些二进制文件，文件都是可以被运行的
  • Dev:该目录中主要存放的是外接设备，如U盘其他的光盘，在其中的外接设备不可能被直接使用，需要挂载（类似win下分配盘符
  • Etc：该目录主要存储一些配置文件
  • Home：表示“家”，表示除了root用户以外的其他用户的家目录，类似于windows下的User/用户目录
  • Proc:表示进程，该目录中存储的是Linux运行时候的进程
  • Sbin：全称super binary，该目录存在一些可被执行的二进制文件，但是必须得有super权限的用户才能执行
  • Tmp：表示“临时”的，当系统运行的时候产生的临时文件会在这个目录中存
  • Usr:用户自己存在文件，相当于windows下的program files.
  • Var：存放的程序/系统的日志文件的目录。
  • Mnt:当外接设备需要挂载的时候，就需要挂到mnt目录下

Linux的基本指令

指令

定义：指在Linux终端（命令行）中输入的内容就称之为指令

一个完整的指令格式： Liunx通用格式：#指令 [选项] 【操作对象】

注：一个指令可以包含多个选项，操作对象也可以是多个

ls指令 (list)

  • # ls ：列出当前目录下的所有文件/文件夹的名称

  • # ls路径 ：列出指定路径下的所有文件/文件夹的名称

    注：关于路径（inportant)

    路径可以分为两种：相对路径、绝对路径。

    相对路径：相对首先要有一个参照物（一般就是当前的工作路径）

    相对路径的写法：在相对路径中通常会用到 “ ./ “【表示当前目录下】【表示上一级的目录下】 , “ ../ “两个符号。

    绝对路径：不需要参照物，直接从根“/”开始寻找相对路径；

  • # ls 选项 路径 ： 在列出指定路径下的文件/文件夹的名称，并以指定的格式进行显示。

    常见的语法：

    ​ # ls -l

    ​ # ls -la

    选项解释：

    -l：表示list，表示以详细列表的形式进行展示

    -a：显示说有的文件/文件夹（包含了隐藏文件/文件夹)

​ 注：上述第一列为文档类型 “-”表示对应文档类型为文件，“d”对应文档类型为文件夹。

​

​ 注：在Linux中隐藏文件一般都是以“ . ”开头。

  • # ls -lh ：列出指定文件下的所有文件/文件夹的名称，以列表的形式并且在显示文档大小的时候一可读性较高的形式显示

pwd指令

用法：#pwd（print working directory,打印当前的工作目录）

cd指令

作用：相当于切换当前的工作目录

语法：#cd路径

案例：当前在“/”下，需要使用绝对路径切换到/usr/local.

案例：当前在/usr/local下，需要使用相对路径切换目录到home目录下的Linux用户目录中去

补充：在liunx中有一个特殊的符号“~”，表示当前用户的家目录切换的方式：# cd ~

mkdir指令

指令：mkdir (make directory,创建目录）

  • 语法1：# mkdir 路径 【路径，可以是文件夹名称也是可以是包含名称的一个完整路径】

    案例： 在当前路径下创建出目录“jingli“

​ 注意：ls 列出的结果颜色说明，其中蓝色表示文件夹，黑色表示文件，绿色 的其权限为拥有所有权。

​ 案例：在指定路径下创建一个文件夹“jinglj”

  • 语法2.：#mkdir -p 路径 : 含义：一次性创建多个

  • 语法3：#mkdir 路径1 路径2 路径3 ……【一次性创建多个目录】

touch指令

指令：touch

作用：创建文件

语法：#touch 文件路径 【路径可以是直接的文件名也可以是路径】

案例：使用touch来创建一个文件，命名为Linux.txt

可以同时创建多个文件

案例：使用touch来在home用户的加目录中创建文件，Liunx.txt。

cp指令

指令：cp （copy)

Effect: copy 文件/文件夹到指定位置。

语法：# cp 被复制文档的路径 文档被复制到的路径

注意：Liunx 在复制过程中可以重新多新的文件进行重命名，但是不是必须需要的，则保持前后名称一致。当使用cp命令进行文件夹复制操作的时候唔要添加选项“-r”【-r表示递归复制】,否则目录将被忽略

mv指令

指令：mv (move,移动,剪切）

作用：移动文档到新的位置

语法：#mv 需要移动的文档路径 需要保持的位置路径

移动后文件夹不在原来的位置

# mv liunx1.txt /linux1.txt

# mv /home/linux/ /

重命名： mv /jingli /weixun :在Linux中重命名命令也是mv,语法和移动语发一样。

rm指令

指令：rm (remove ,移动，删除）

作用：移除/删除文档

语法：#rm 选项 需要移动的文档路径

选项：

​ -f:强制删除，不需要输入提示 rm -f /Linux1.txt

​ -r :表示递归

​ -rf:移除目录

删除一个目录的时候需要做递归删除，并且一般也不需要进行删除确认询问，一般用-rf选项

删除一个文件里有公共特性的文件 # rm -f linu*

其中 “*”表示通配符，意思表示任意的字符。

vim指令

指令： vim (vim是一款文本编译器）

语法： #vim 文件的路径

作用： 打开一个文件。

推出打开的文件：在没有按下其他命令的时候，按下shift+英文冒号，输入q，按下回车键即可

输出重定向

一般的命令输出在终端中，有些时候需要将一些命令的执行结果保存到文件中进行分析，则用到输出重定向。

>: 覆盖输出，会覆盖原先的文件内容

>>:追加输出，不会覆盖掉原先的文件内容，会在原始内容末尾继续添加

语法： # 正常执行的指令 >/>>文件的路径

注意：文件可以不存在，不存在则新建

cat指令

作用1：直接打开一个文件的指令

语法：cat 文件的路径

作用2：可以对文件进行合并

语法2：#cat 带合并的文件路径1 带合并的文件路径2… …文件路径n > 合并后的文件路径

Linux的进阶指令

df指令

作用：查看磁盘空间

语法：# df -h -h：表示以可读性较高的形式展示大小

free指令

作用：查看内存使用情况

语法：# free -m -m:表示以mb为单位进行查看
